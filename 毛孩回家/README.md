# 🐾 毛孩回家 — 流浪动物识别匹配小程序

> 一个微信小程序，帮助捡到流浪动物的人与走丢宠物的主人建立联系，也提供一个领养平台。

---

## 📋 功能总览

```
首页
 ├── 发布捡到 → 拍照上传 → 系统匹配走丢宠物 → 查看匹配结果
 ├── 发布丢失 → 上传照片 → 系统匹配捡到动物 → 查看匹配结果
 ├── 走丢浏览 → 像淘宝那样按特征筛选浏览
 ├── 捡到浏览 → 分类筛选浏览
 ├── 领养区 → 浏览待领养动物
 └── 个人中心 → 管理我的发布
```

## 🚀 从零开始上手指南

### 第1步：注册微信小程序

1. 打开 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击「立即注册」→ 选择「小程序」
3. 用邮箱注册，完成实名认证（需要个体户或企业资质，或个人主体）
4. 登录后，在「开发」→「开发管理」→「开发设置」中获取 **AppID**
5. 记下 AppID，后面要用

### 第2步：开通云开发

1. 在微信公众平台左侧菜单点「开发」→「云开发」
2. 点「开通」，按提示开通云环境
3. 创建成功后，记下 **云环境ID**（如 `pet-match-123abc`）
4. 点「数据库」→ 创建以下集合：

| 集合名 | 权限设置 |
|--------|---------|
| `found_animals` | 所有用户可读，仅创建者可写 |
| `lost_pets` | 所有用户可读，仅创建者可写 |

> 权限设置方式：数据库 → 对应集合 → 权限设置 → 选择"所有用户可读，仅创建者可写"

### 第3步：下载安装开发者工具

1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 安装后打开，用微信扫码登录

### 第4步：导入项目代码

1. 在开发者工具中点击「导入项目」
2. **AppID**：填入第1步获取的 AppID
3. **项目目录**：选择本项目的 `miniprogram` 文件夹（或整个 `pet-match` 文件夹）
4. 点击「确定」

### 第5步：配置云开发

在项目中修改两处配置：

**文件1：`miniprogram/app.js`**（第44行附近）
```javascript
wx.cloud.init({
  env: '你的云环境ID',  // ← 改成第2步记下的云环境ID
  traceUser: true
})
```

**文件2：上传页面的云存储路径**（如果有云函数，也需要配置）

### 第6步：上传并部署云函数

在开发者工具中：
1. 左侧点「云开发」图标
2. 右键 `cloudfunctions` 文件夹下的每个云函数
3. 点击「上传并部署：云端安装依赖」
4. 等待所有云函数部署完成

需要部署的云函数：
| 云函数 | 功能 |
|--------|------|
| `getMatchList` | 智能匹配算法 |
| `uploadAnimal` | 上传动物信息 |
| `getAnimalDetail` | 获取详情 |
| `getAdoptList` | 领养列表 |
| `searchAnimals` | 搜索动物 |

### 第7步：在手机上预览

1. 开发者工具中点击「预览」
2. 用微信扫码即可在手机上体验
3. 也可以点击「真机调试」进行调试

## 🧠 匹配算法原理

匹配系统会从 **6个维度** 计算两只动物的相似度：

| 维度 | 权重 | 说明 |
|------|------|------|
| 颜色 | 30分 | 同色满分，近似色部分得分（如 白≈灰） |
| 地点 | 25分 | 越近分越高，1公里内满分 |
| 体型 | 20分 | 同体型满分，相邻体型部分得分 |
| 特征 | 15分 | 项圈、性别、绝育情况综合 |
| 时间 | 10分 | 捡到和走丢时间越近分越高 |

匹配结果分为四个等级：
- 🟢 **高度匹配**（≥80%）- 极有可能是同一只
- 🔵 **可能匹配**（60-79%）- 有较大可能是
- 🟠 **低度匹配**（40-59%）- 有一定可能，需要人工比对
- ⚪ **不匹配**（<40%）

## 🎨 特征分类体系（像淘宝服装分类）

本系统将动物特征按维度分类，方便筛选浏览：

```
动物类型 → [猫] [狗]

颜色 → [白色] [黑色] [黄色] [灰色] [花色] [棕色] [奶牛色] [狸花色]

体型 → [小型 ≤5kg] [中型 5-15kg] [大型 ≥15kg]

性别 → [公] [母] [未知]

项圈 → [有] [无] [不确定]
  └─ 有项圈 → 项圈颜色：[红] [蓝] [黑] [绿] [黄] [花色]

绝育 → [已绝育] [未绝育] [不确定]
```

## 💡 常见问题

### Q：我没有服务器怎么办？
A：使用了微信云开发，不需要自己买服务器。数据库、存储、云函数都是微信提供的，每个月有免费额度。

### Q：我不会写代码，怎么修改？
A：项目代码已经全部写好，你只需要：
1. 在微信开发者工具中打开项目
2. 修改 `app.js` 里的云环境ID
3. 修改 `project.config.json` 里的 AppID
4. 部署云函数
5. 预览使用

### Q：怎么把项目变成自己的小程序？
A：注册自己的小程序账号 → 获取自己的 AppID → 替换项目中的 AppID → 上传代码 → 提交审核 → 发布。
